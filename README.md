> Build real progress.

Vexor é um aplicativo de treino focado em consistência, simplicidade e progresso real.

O objetivo do app é ajudar usuários a acompanhar seus treinos, manter constância e visualizar evolução de forma clara e direta.

---

## 🚀 Features

- 📊 Acompanhamento de treinos
- 🔥 Streak de consistência
- 🏋️ Criação de treinos personalizados
- 📈 Progresso semanal
- 🧠 Interface minimalista e intuitiva
- 📱 Design focado em performance e usabilidade

---

## 🎨 Design

O Vexor segue um estilo:

- Minimalista (preto, branco e tons neutros)
- Foco em tipografia forte
- UI limpa e moderna
- Experiência direta (sem distrações)

---

## 🧱 Tecnologias

### Frontend
- Flutter

### Backend (planejado)
- Supabase (PostgreSQL)
- API REST

---

## 🧠 Conceito

O Vexor não é apenas um app de treino.

A ideia central é:

"Consistency beats motivation."

Foco em:
- constância > intensidade
- progresso real > estética
- simplicidade > complexidade
